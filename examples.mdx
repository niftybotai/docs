NiftyBot is a universal enrichment engine. The snippets below highlight real-world use cases that balance speed, accuracy, and clever reasoning. Each example uses `field_type` or `instructions` only when they add clarity.

## 1. Quick CRM Lookup (Simple)
```bash
curl -X POST https://api.niftybot.ai/v1/enrich \
  -H "Content-Type: application/json" \
  -H "X-API-Key: $NIFTYBOT_API_KEY" \
  -d '{
    "source": {
      "company": "Microsoft",
      "title": "Chief Executive Officer"
    },
    "requests": [
      { "field_name": "full_name" },
      { "field_name": "linkedin_url" },
      { "field_name": "previous_company" }
    ]
  }'
```
```json
{
  "request_id": "req_01J4G0R3SYEHJW0W7Z9TSBY6KX",
  "enrichments": [
    { "field_name": "full_name", "value": "Satya Nadella" },
    { "field_name": "linkedin_url", "value": "https://www.linkedin.com/in/satyanadella/" },
    { "field_name": "previous_company", "value": "Sun Microsystems (software engineer, early career)" }
  ]
}
```
